# Личный финансовый ассистент

Учебный проект по курсу **«Введение в специальность»**.

Проект представляет собой консольное приложение на Python для учёта личных финансов. Пользователь может добавлять доходы, расходы и финансовые события, а программа сохраняет данные, анализирует бюджет и выводит рекомендации.

## Описание проекта

Личный финансовый ассистент помогает пользователю контролировать свои доходы и расходы.  
Программа предназначена для простого учёта личного бюджета без сложного интерфейса и базы данных.

Пользователь запускает приложение в консоли, выбирает нужное действие в меню и вводит данные: сумму дохода, расхода, категорию или финансовое событие. Все записи сохраняются в JSON-файл и могут использоваться для дальнейшей аналитики.

Приложение показывает общий доход, общий расход, баланс, распределение расходов по категориям, а также может дать рекомендации на основе введённых данных.

## Постановка задачи

Проблема, которую решает проект: обычному пользователю неудобно вручную отслеживать свои доходы, расходы и финансовые планы. Из-за этого сложно понять, куда уходят деньги, сколько остаётся в распоряжении и на каких категориях можно экономить.

Задача проекта — разработать программное обеспечение, которое позволяет:

- добавлять доходы;
- добавлять расходы;
- указывать категории расходов;
- добавлять события и финансовые планы;
- сохранять данные пользователя;
- анализировать финансовые записи;
- выводить рекомендации по управлению бюджетом.

## Тип разрабатываемого ПО

Тип программного обеспечения: **console application / консольное приложение**.

Такой формат выбран потому, что он подходит для учебного проекта и позволяет сосредоточиться на основной логике программы: ввод данных, сохранение, обработка и вывод результата.

## Как запустить проект

1. Склонировать репозиторий:

```bash
git clone https://github.com/Ofrai-pro/finance-ai.git
```

2. Перейти в папку проекта:

```bash
cd finance-ai
```

3. Установить зависимости:

```bash
pip install -r requirements.txt
```

4. Запустить программу:

```bash
python main.py
```

## Настройка нейросети

Основные функции программы работают без API-ключа: можно добавлять доходы, расходы, события и смотреть аналитику.

Для работы пункта **«Нейросеть (Gemini через OpenRouter)»** нужно получить API-ключ OpenRouter и указать его в файле `src/ai_assistant.py` вместо заглушки:

```python
api_key="ТВОЙ_OPENROUTER_КЛЮЧ"
```

API-ключ не должен храниться в публичном репозитории.

## Как пользоваться программой

После запуска пользователь видит меню:

```text
===== Финансовый ассистент =====
1. Добавить доход
2. Добавить расход
3. Добавить событие/план
4. ИИ-помощник (аналитика и правила)
5. Нейросеть (Gemini через OpenRouter) – получить совет
6. Выход
```

Пользователь выбирает пункт меню и вводит необходимые данные.

Пример добавления дохода:

```text
Сумма: 13000
Источник: стипендия
Комментарий: за май
```

Пример добавления расхода:

```text
Сумма: 750
Категория: еда
Комментарий: продукты
```

Пример добавления события:

```text
Описание: Покупка ноутбука
Дата начала: 2026-07-01
Дата окончания: 
Плановая сумма: 60000
```

После добавления записей можно открыть аналитику. Программа покажет доходы, расходы, баланс и распределение трат по категориям.

Данные не обнуляются каждый месяц. Программа хранит все записи постоянно и анализирует сохранённые данные.

## Безопасность

В проекте учтено, что API-ключи нельзя хранить в открытом доступе. Поэтому настоящий ключ OpenRouter не должен попадать в GitHub.

Также файл с пользовательскими финансовыми данными `data/records.json` не загружается в репозиторий, так как он содержит личные записи пользователя.
